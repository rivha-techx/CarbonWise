# CarbonWise

CarbonWise is an AI-powered platform that helps users understand, track, and reduce their carbon footprint through personalized insights and sustainable lifestyle recommendations.

## Features

- Carbon Footprint Calculator
- Personalized AI Recommendations
- Sustainability Challenges
- Progress Tracking Dashboard
- Eco-Friendly Habit Building

## Problem Statement

Design a solution that helps individuals understand, track, and reduce their carbon footprint through simple actions and personalized insights.
